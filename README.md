## Descripción
Implementación de un sistema de balanceo de carga usando Nginx como proxy inverso
con 3 servidores backend en AWS EC2.

## Estructura
```
├── configs/          # Archivos de configuración
├── scripts/          # Scripts de automatización
├── docs/             # Documentación adicional
└── evidencias/       # Capturas y logs
```

## Tecnologías
- Nginx 1.18+
- Node.js 18+
- Ubuntu 22.04
- AWS EC2
- SSL/TLS (Let's Encrypt)
